# Medical Phrases PWA

Это готовое PWA-приложение: сайт с карточками, поиском, фильтрами по книге/главе/приоритету и локальным прогрессом.

Внутри сейчас: 2000 фраз.

## Как проверить на Windows

1. Распакуйте ZIP.
2. Откройте папку `MedicalPhrasesPWA`.
3. Дважды нажмите `start_local_server.bat`.
4. В браузере откройте: http://localhost:8000

Через `file://` приложение может не работать корректно, потому что браузер блокирует загрузку JSON и service worker.

## Как поставить на iPhone как иконку

Для iPhone нужен сайт по HTTPS. Самый простой путь без программирования:

1. Зарегистрируйтесь на Netlify.
2. Откройте раздел Deploys.
3. Перетащите туда всю распакованную папку `MedicalPhrasesPWA`.
4. Netlify выдаст HTTPS-ссылку.
5. Откройте эту ссылку на iPhone именно в Safari.
6. Нажмите кнопку Share / Поделиться.
7. Выберите Add to Home Screen / На экран «Домой».
8. После первой загрузки приложение будет работать офлайн.

## Файлы

- `index.html` — главный экран.
- `app.js` — логика карточек, фильтров и прогресса.
- `data/phrases.json` — база фраз.
- `manifest.webmanifest` — параметры установки.
- `service-worker.js` — офлайн-кэш.
- `icons/` — иконки приложения.

## Важно

Прогресс хранится в `localStorage` конкретного браузера/устройства. Если очистить данные Safari, прогресс исчезнет. Кнопка «Экспорт прогресса» сохраняет CSV-файл со статусами.
